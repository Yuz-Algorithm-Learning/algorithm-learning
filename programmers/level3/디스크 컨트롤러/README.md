/* {작업의 소요시간, 작업 요청 시점}: 작업의 소요 시간이 적은 순으로 정렬(오름차순) */

### 풀이
```c++
- 중요 기준: 작업의 소요 시간이 적은 순서대로 다다다닥!!
- 한 작업을 진행할 때마다 작업이 요청되는 시점이 end_time 보다 작거나 같은 작업들 중, 작업 소요시간이 가장 작은 것을 추출!
- 큐에 들어있는 게 없다면 다음 작업 중 가장 작은 작업을 기준으로 다시 시작!
    - (0, 3), (4, 1), (4, 2) 일 때
        - (0, 3) 에 대한 작업을 진행 한 후, 큐가 비어있음
        - lines 32~34: (4, 1) 기준으로 작업 끝난 시점(end_time)을 설정하고, 총 소요 시간(total_time) 계산
```
