## 풍선 터뜨리기

### 풀이
1. 타겟 기준으로 왼쪽 중의 최솟값, 오른쪽 중의 최솟값을 찾아
2. 타겟이 왼쪽 최솟값과 오른쪽 최솟값보다 클 때 풍선을 터뜨릴 수 없음

- 시간 초과 ...
    - 해결: 최소값을 그때그때 구하지말고 구해놓자!!!
    - [예시] [-16,27,65,-2,58,-92,-71,-68,-61,-33]
        - `-> 방향`: [-16, -16, -16, -16, -16, -92, -92, -92, -92, -92]
        - `<- 방향`: [-92, -92, -92, -92, -92, -92, -71, -68, -61, -33]
