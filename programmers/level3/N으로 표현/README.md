## N으로 표현

### 변수
```c++
- 다이나믹 프로그래밍
- dp[n]: n개의 N을 사용했을 때 나오는 값들 저장
    - N이 5일 때  
        - 초기 설정  
            - dp[1] = 5  
            - dp[2] = 55, 5 + 5, 5 - 5, 5 * 5, 5 / 5  
        - 이후  
            - dp[3] = (dp[1], dp[2] 에 대한 사칙연산 수행 한 값), (dp[2], dp[1] 에 대한 사칙연산 수행 한 값)  
            - dp[4] = (dp[1], dp[3]), (dp[2], dp[2]), (dp[3], dp[1]) 에 대한 사칙연산 수행 한 값   
            ...
- unordered_map 을 이용해 중복값 처리
```
