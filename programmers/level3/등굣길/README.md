## 등굣길

### 풀이
```c++
- 주의: input 이 (y, x) 형태가 아닌 (x, y) 형태로 들어옴!

- lines 26~37: (1, 1) ~ (1, m) 과 (1, 1) ~ (n, 1) 을 1로 채워줌
    - 이때, 채워주다가 웅덩이를 발견한다면 그 다음 좌표부터는 갈 수 없는 길이므로 break
- lines 39~48: 현재 좌표의 위쪽과 왼쪽의 값을 더해서 채워줌
    - 웅덩이가 있는 부분은 0으로 대체!
    - 계산할 때마다 1000000007 로 나누어주기 !!
```
