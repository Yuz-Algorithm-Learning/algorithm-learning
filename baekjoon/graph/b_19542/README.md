## 19542 전단지 돌리기
```
알고리즘: DFS
```

### 풀이
```
1. visit 배열: 방문 유무
2. root 배열: 갈라지는 노드 부분의 가지 중 가장 긴 깊이의 가지 깊이를 넣어줌 (DFS 이용)
3. 케니 소프트의 위치인 S를 제외한, 힘 D보다 크거나 같은 노드의 개수를 더함
4. 왕복이므로 각 노드의 2배를 적용
```
