## 1987 알파벳

### 시도
```
* 방문한 알파벳을 나타내기 위해 벡터로 만들어 push & pop을 사용했지만, 이는 확인할때마다 이 벡터에서 찾아야 했음
* 해결: visit_alpha[26] 의 배열을 만들어 아스키코드를 활용해 A->0, B->1 과 같은 방식으로 저장
```

### 풀이
```
1. 방문한 알파벳을 visit_alpha에 저장
2. 해당 노드를 기준으로 '오른쪽, 아래, 왼쪽, 위' 순서로 방문을 안했다면 지나가도록 구현
3. 네 가지 방향을 다 검사했다면,   
    해당 노드를 기준으로 최대 칸 수를 저장  
    해당 노드(알파벳)를 visit_alpha에서 제거 및 칸 수 하나 줄이기 
```
