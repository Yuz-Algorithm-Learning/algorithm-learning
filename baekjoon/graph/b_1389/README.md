## 1389 케빈 베이컨의 6단계 법칙

### 알고리즘
```
 - 플로이드 알고리즘
 - 점 {1, 2, ... , k} 만을 경유 가능한 점들로 고려하여,  
   점 i로부터 점 j까지의 모든 경로 중에서 가장 짧은 경로의 거리
 -  map[i][j] = min(map[i][k] + map[k][j], map[i][j]);
```
