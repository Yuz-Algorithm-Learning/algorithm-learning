## 2616 소형 기관차

### 배열
```
- acc[i]: i번째까지의 누적 손님 수  
- dp[i][j]: i번째 소형기관차가 j번째 객차까지 봤을 때의 최대로 운송할 수 있는 손님 수 
```

 ### 풀이
 ```
1. 손님을 입력받을 때마다 누적하여 acc 배열에 저장
2. 각 기관차의 범위는 [i * M, N - M * (3 - i)] 적용  
    - 이때 i는 소형 기관차 번호, M은 최대로 끌 수 있는 객차 수  
3. dp[i][j] 에는 현재 객차(j)를 포함했을 떄와 안 했을 때의 방식 중 큰 값 저장  
    - 포함 X: j-1번째 객차까지의 최대 운송할 수 있는 손님 수  
    - 포함 O: i-1번째 소형기관차가 j-m번째 객차까지 봤을 때 최대 운송할 수 있는 손님 수   
              + i번째 소형기관차가 (j-m, j] 객차까지 봤을 때 최대 운송할 수 있는 손님 수   
4. 소형 기관차는 총 3개이므로, 3번째 소형기관차가 N번째 객차까지 봤을 때의 최대 운송할 수 있는 손님 수를 구해야함  
    - dp[3][N] 적용
 ```
