## 12907 동물원

### 풀이
```
1. 동물은 총 N마리  
   -> 두번째 줄에 입력되는 숫자는 N-1 보다 큰 숫자가 나올 수 없음
2. 동물은 토끼와, 고양이 두 종류로 이루어져 있음  
   -> 두번째 줄에 입력되는 숫자 중 같은 숫자는 3개 이상 나올 수 없음
3. 위에 대한 예외처리 후, 입력받는 숫자에 해당하는 index에 동물의 수를 높여줌
4. 각 동물은 연속적으로 이루어져 있어야함  
   Ex) 5마리라면, 입력: 2 0 1 0 1 | animal 배열: 2 2 1 0 0
5. animal 배열에 2개씩 존재하면 2배를 적용  
   -> 그 외에 1개씩 나오는 경우엔 같은 동물이므로, 맨 마지막에 2배를 적용
```
