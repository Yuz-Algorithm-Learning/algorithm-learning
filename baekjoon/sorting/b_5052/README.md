## 5052 전화번호 목록

### 풀이
```
1. 입력받는 전화번호 목록을 string 기준으로 sort 진행  
    Ex) input: 9110, 911, 97625999, 91125426  
        sort: 911, 9110, 91125426, 97625999   
2. i 번째 번호와 i+1 번째 번호를 비교했을 때, i 번째 번호가 i+1 번째 번호의 접두어라면 i+1 번째 번호는 i 번째 번호의 길이 만큼을 포함하고 있음
3. 따라서 i+1 번째 번호가 i 번째의 번호를 포함하고 있는지를 확인
```
